HTML


IDEAS GENERALES

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y estructurar el contenido en la web. Básicamente, HTML proporciona la base de una página web, definiendo elementos como párrafos, encabezados, imágenes, enlaces, tablas, y más. Estos elementos se representan mediante "etiquetas" que indican cómo debe aparecer y comportarse cada parte del contenido.

APUNTES

HTML dice el profe es la piel de un monito de lego, yo diría que es la pluma para una hoja cotidiana, simple y claro.



RESUMEN

HTML es lo que indica el ordenado de las páginas, es la estructura de esta contando con los textos, imágenes y su contenido.


CSS


IDEAS GENERALES

Cascading Style Sheets o CSS es la separacion del contenido y el estilo en donde se puede modificar la apariencia de una pagina web html. CSS se encarga del diseño visual, como colores, fuentes, márgenes, y disposición de los elementos.

APUNTES

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación de un documento escrito en HTML o XML.


RESUMEN

En CSS enchula el HTML, le da mejor aspecto visual.


ETIQUETA A


IDEAS GENERALES

La etiqueta en HTML sirve para crear hipervínculos o enlaces, que permiten a los usuarios navegar de una página a otra, ya sea dentro del mismo sitio web o hacia una dirección externa. Es una de las etiquetas más esenciales para la navegación web.

APUNTES

Sirve para ligar documentos externos al código HTML


RESUMEN

Sirve para ligar documentos externos al código HTML


ETIQUETA IMG


IDEAS GENERALES

Sirve para insertar imágenes (src) en una página, también debe de contar con una descripción clara (alt) y si es necesario una fuente de donde se obtuvo la imagen. Más allá del código las imágenes deben estar guardas en una carpeta aparte dentro de la carpeta del html.

APUNTES

Las imágenes deben estar guardadas con un nombre corto y fácil de ubicar, deben de tenerse guardadas en el menor tamaño posible lo cual podemos conseguir de forma rápida en Photoshop.


RESUMEN

Mediante la etiqueta img conseguimos insertar una imagen en la web.


TEXTO


IDEAS GENERALES

El texto en una página web es el contenido escrito que transmite información a los usuarios. En HTML, se utiliza una variedad de etiquetas para estructurarlo correctamente.

APUNTES

Sirven para los textos de los párrafos resaltar ciertas palabras, el profe hablo de un ejemplo sobre el cambio de fecha de un concierto para una pagina en donde rayaba el texto previo para aclarar que la fecha inicial ya no contaba.

  • b> texto en negritas
  • strong> texto en negritas
  • i> texto en italica
  • em> texto en enfasis
  • mark> texto marcado
  • small> texto pequeno
  • del> texto tachado
  • ins> texto insertado
  • sub> texto debajo del renglón
  • sup> texto superior


RESUMEN

Resalta textos o los modifica de forma particular.


PROCESO DE DISENAR


IDEAS GENERALES

  • Planificación: Definir el propósito y objetivos, Conocer la audiencia, Investigación de competidores y tendencias, Estructura del contenido (mapa del sitio)
  • Wireframing y prototipado: Wireframe, Prototipo
  • Diseño visual: Elegir un estilo visual, Diseñar la interfaz, Diseño responsive.
  • Desarrollo: HTML (estructura), CSS (estilos), JavaScript (interactividad), Gestión de contenidos (CMS).
  • Pruebas Compatibilidad: de navegadores, Pruebas en dispositivos, Velocidad de carga, Accesibilida.
  • Lanzamiento Hosting, Dominio, SEO básico.
  • Mantenimiento: Actualización de contenido, Monitoreo, Seguridad.

APUNTES

No se diseña una página web durante su codificación, esta debe de estar diseñada desde antes



RESUMEN

En CSS enchula el HTML, le da mejor aspecto visual.
